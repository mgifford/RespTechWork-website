## Library events

We invite you to **use the library for your own conversations on responsible tech**. Please keep in mind that we're currently using the free hosted version of [WorkAdventure](https://workadventu.re/) with a limit of 15 concurrent users. If you're meeting in a larger group, you can use the large table in the garden, where visitors can join a Jitsi meet room.
